A helpful method to check to see if a transform is facing another transform in Unity3d

{% highlight csharp %} private bool IsFacing(Transform target) { Vector3 forward = transform.TransformDirection(Vector3.forward); Vector3 toTarget = target.position - transform.position; return Vector3.Dot(forward, toTarget) > 0; } {% endhighlight %}
